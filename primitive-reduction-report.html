변경 근거 (무엇을 보고 결정했나)

  • 참조 그래프 분석 — 모든 변수의 mode 값을 훑어 VARIABLE_ALIAS 관계를 집계. "어떤 변수도 가리키지 않는 primitive"를 죽은 토큰으로 판정 → 202/310.
  • 팔레트 출처 추정 — 16개 유채색 family가 모두 동일한 100~1600 16단계. 이는 Adobe Spectrum 전체 팔레트를 통째로 들여온 흔적. 실제 브랜드는 오렌지 1종.
  • 안전 우선 삭제 — 8개 이색 family로 대상을 화이트리스트 한정하고, 그 안에서도 참조가 0인 토큰만 삭제. 참조 걸린 step은 자동 보존(=alias 안 깨짐).
  • 단계적 정리 — 컴포넌트 484개가 primitive를 직접 참조 중이라, 기능색 ramp 축소는 재배선 이후가 더 안전 → 이번엔 보류.
판정 기준은 변수→변수 참조입니다. 캔버스 노드에 직접 칠해진 사용은 이번 범위 밖이며, 설령 있었더라도 삭제 시 색은 raw 값으로 남아 시각 변화는 없습니다(토큰 연결만 해제).
